# Pass It On

A full-stack travel journal application where users can select, customize and share card templates for different occassions.

## Live link 

https://pass-it-on-zfmi.vercel.app/

## Tech Stack

**Frontend:** React, React Router, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB, Mongoose  

## Features

- Browse and select card templates for different occasions
- Customize card content
- Share cards with others

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository
```bash
git clone https://github.com/UzzmaShahid/PassItOn.git
```

2. Install dependencies for both client and server
```bash
cd Client
npm install

cd ../Server
npm install
```

3. Create a `.env` file in the Server folder and set the environment variables
4. Run the development servers
```bash
# Run backend
cd Server
node index.js 

# Run frontend
cd Client
npm run dev
```
